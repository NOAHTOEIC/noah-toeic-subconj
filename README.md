# 노아토익 부사절접속사 완전 정복

토익 시험 대비 **부사절접속사 · 전치사 · 접속부사** 암기 웹앱입니다.

## 주요 기능

### 학습 카드 (플래시카드)
- **부사절접속사** — 시간 · 조건 · 양보 · 이유 · 결과 · 기타 (6개 카테고리)
- **전치사** — 시간 · 조건 · 양보 · 이유 · 기타 (5개 카테고리)
- **전치사 & 부사절접속사 겸용** — Until, Since, As, After, Before, Considering
- **접속부사** — 내용추가 · 구체화 · 예시 · 반대 · 인과관계 · 시간순서 · 조건 · 강조 · 결론 · 기타 (10개 카테고리)
- 카드를 탭하면 **뜻 + 품사 배지** 표시
- 겸용 단어는 어떤 카테고리에서든 **전치사 & 부사절접속사 겸용**으로 정확히 표기

### 퀴즈 (4종류)
- 부사절접속사 뜻 맞추기
- 전치사 뜻 맞추기
- 접속부사 뜻 맞추기
- 품사 구분 퀴즈 (A.전치사 B.부사절접속사 C.겸용 D.접속부사 — 고정 순서)
- 틀린문제 모음집 (자동 저장 & 복습)

### 기타 기능
- 스와이프로 카드 넘기기
- 북마크 저장 (품사 정보 포함)
- 학습 진도 추적
- PWA 지원 (홈 화면에 추가 가능, 오프라인 사용)

## GitHub Pages 배포 방법

1. 이 저장소를 GitHub에 Push
2. **Settings → Pages → Source** 에서 `main` 브랜치 선택
3. 저장하면 `https://[username].github.io/[repo-name]/` 에서 접속 가능

## 파일 구조

```
├── index.html      # 메인 앱 (HTML + CSS + JS 단일 파일)
├── manifest.json   # PWA 매니페스트
├── sw.js           # 서비스 워커 (오프라인 지원)
├── icon-192.png    # PWA 아이콘 (192x192)
├── icon-512.png    # PWA 아이콘 (512x512)
└── README.md       # 이 파일
```

## 모바일 앱처럼 사용하기

### iPhone (Safari)
1. 사이트 접속 → 공유 버튼 (□↑) 탭
2. **"홈 화면에 추가"** 선택
3. 홈 화면에서 앱처럼 실행

### Android (Chrome)
1. 사이트 접속 → 메뉴 (⋮) 탭
2. **"홈 화면에 추가"** 또는 **"앱 설치"** 선택
3. 홈 화면에서 앱처럼 실행
